Database Class

Properties
name

The database’s name.

path

The database’s path.

count

The number of documents in the database.

config

The database’s configuration.

Methods
-initWithName:error:

Initializes a database object with a given name and the default database configuration.

-initWithName:config:error:

Initializes a Couchbase Lite database with a given name and database configuration.

-init

Not available.

-documentWithID:

Gets an existing CBLDocument object with the given ID.

-contains:

Checks whether the document of the given ID exists in the database or not.

-objectForKeyedSubscript:

Gets a document fragment with the given document ID.

-saveDocument:error:

Saves the given document to the database.

-deleteDocument:error:

Deletes the given document.

-purgeDocument:error:

Purges the given document from the database.

-inBatch:do:

Runs a group of database operations in a batch.

-close:

Closes a database.

-deleteDatabase:

Deletes a database.

-compact:

Compacts the database file by deleting unused attachment files and vacuuming the SQLite database.

-changeEncryptionKey:error:

Changes the database’s encryption key, or removes encryption if the new key is nil.

+deleteDatabase:inDirectory:error:

Deletes a database of the given name in the given directory.

+databaseExists:inDirectory:

Checks whether a database of the given name exists in the given directory or not.

-addChangeListener:forDocumentID:

Add a document change listener to the document.

-removeChangeListener:forDocumentID:

Remove the document change listener from the document.

-allDocuments

Enumerates all documents in the database, ordered by document ID.

-createQueryWhere:

Compiles a database query, from any of several input formats.

-createIndexOn:error:

Creates a value index (type kCBLValueIndex) on a given document property.

-createIndexOn:type:options:error:

Creates an index on a given document property.

-deleteIndexOn:type:error:

Deletes an existing index.
